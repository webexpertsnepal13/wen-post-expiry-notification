== Description ==

Set expiry date for posts, pages, and custom post types. Send notification about the expiry before specified number of days.

Note: Select your appropriate timezone on Settings > General > Timezone.

Features:
- Set expiry date for any posts and custom post types
- Ability to change email option for individual posts
- Send notification about expiry before allocated days
- Ability to change time when notification should be sent
